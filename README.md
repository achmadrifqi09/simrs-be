## Description

Backend repository of the Hospital Management System of the University of Muhammadiyah Malang

## Project setup

```bash
$ npm install
```

```bash
$ npx prisma migrate dev
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
